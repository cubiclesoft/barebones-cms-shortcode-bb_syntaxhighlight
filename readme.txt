Barebones CMS Syntax Highlighter Shortcode
(C) 2014 CubicleSoft.  All Rights Reserved.

Description:  Adds a new shortcode to the content editor that makes it easy to
              embed highlighted source code.

Leverages Alex Gorbatchev's third-party library SyntaxHighlighter (ver 2.1.382).
(http://alexgorbatchev.com/wiki/SyntaxHighlighter)


Installation:

Upload the 'support' and 'widgets' subdirectories to your Barebones CMS
installation.

Click "Configure Security" from within the content widget options list to set
up the new shortcode's security options.
